Project Freud(e)

Project Freud(e) is a project to gamify psychological experiments. The goal is to raise ecological validity and to make research accessible to more researcher. This is a fully independent project with no associations. The collection of data doesn't rely on funding. This app is heavily reliant on jsPsych. I thank all the contributors to jsPsych.

This app is still under heavy development. This is the initial version 0.0.1.
